# SPARK125

For Uniden Bearcat **BC125AT** / **UBC125XLT** / **UBC126AT** / **AE125H** scanners.

### About
**SPARK125** is a collection of **tools**, (developer) **documentation** and **software** to be used with the above mentioned **Uniden Bearcat** scanners.
